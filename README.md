## Featured Project

### KawnGraph

**Give agents the map, not the repo.**

KawnGraph scans a software project into a layered graph and serves precise Context Packs through CLI, MCP, and Studio.

- Local-first by default
- No telemetry
- No cloud dependency for scan or retrieval
- Read-only MCP server
- Evidence-backed relationships
- Built for multiple coding agents

Repository: https://github.com/xd7fx/kawngraph
