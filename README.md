## Mission Control

I like building systems where AI is not just a demo. It has to touch something real: a drone mission, a football match, a store page, an ancient inscription, a student learning robotics, or a developer trying to understand a large codebase.

```text
AI agents        → SABAQ, KawnGraph, automation workflows
Robotics         → YAMAMAH Rescue Drone, ROS, Jetson Nano, Arduino
Computer vision  → YOLOv8, OpenCV, victim detection, lane detection, heritage recognition
Data products    → V-TAC, forecasting, match analytics, dashboards
Commerce systems → Lamha, video commerce, merchant analytics, add-to-cart flows
Education        → Drone 360°, workshops, mentoring, course experiences
```
